# ⏳ Pomodoro Timer (Python)

A simple and effective Pomodoro Timer built using Python.
This project helps improve productivity by breaking work into focused intervals followed by short breaks.

---

## 🍅 What is Pomodoro?

The Pomodoro Technique is a time management method where you:

* Work for **25 minutes**
* Take a **5-minute break**
* Repeat the cycle
* After a few cycles, take a longer break

---

## 🎮 Features

* 25-minute work sessions
* 5-minute short breaks
* 20-minute long break after multiple sessions
* Countdown timer display
* Session tracking (✔ marks or counters)
* Automatic switching between work and break sessions

---

## 🧠 Concepts Used

This project is built using **Object-Oriented Programming (OOP)** concepts:

* **Classes & Objects** for timer logic
* **Encapsulation** to manage session states
